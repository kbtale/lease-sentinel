    Sentinel schema - the core domain entity representing a lease deadline to track.

    Remarks

    A Sentinel represents a single deadline extracted from a lease clause. When the triggerDate arrives, the system dispatches a webhook to the configured URL and updates the status to FIRED.

    Row-Level Security: The userId field is used to filter queries, ensuring users can only access their own sentinels.

    Example

    import { SentinelSchema, Sentinel } from "@/models/schema";

    const data = {
      userId: "user@example.com",
      eventName: "Lease Renewal Deadline",
      triggerDate: "2025-06-15",
      originalClause: "Tenant must provide 90 days notice...",
      webhookUrl: "https://hooks.slack.com/...",
    };

    const result = SentinelSchema.safeParse(data);
    if (result.success) {
      const sentinel: Sentinel = result.data;
    }
