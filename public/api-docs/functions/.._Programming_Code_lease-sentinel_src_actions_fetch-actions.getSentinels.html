      Fetches all sentinels for the authenticated user.

      Returns Promise<
          {
              id?: string;
              userId: string;
              eventName: string;
              triggerDate: string;
              originalClause: string;
              webhookUrl: string;
              status: "PENDING"
              | "FIRED";
              createdAt: Date;
          }[],
      >

      Promise resolving to array of Sentinel objects belonging to the current user, or empty array if:

      • User is not authenticated
      • No sentinels exist
      • Database error occurs

      Security

      Authentication Required: Returns empty array for unauthenticated requests.

      Row-Level Security: Queries are filtered by userId (user's email) to ensure users can only access their own sentinels. This filtering happens at the database query level, not post-fetch, for efficiency and security.

      Remarks

      This function handles Firestore-specific serialization internally:

      • Converts Firestore Timestamp objects to JavaScript Date objects
      • Ensures all returned data is JSON-serializable for React Server Components

      Results are ordered by createdAt descending (newest first).

      Example

      // Server Component usage
      import { getSentinels } from "@/actions/fetch-actions";

      export default async function Dashboard() {
        const sentinels = await getSentinels();

        return (
          <ul>
            {sentinels.map((s) => (
              <li key={s.id}>{s.eventName} - {s.triggerDate}</li>
            ))}
          </ul>
        );
      }
