    Interface ActionState<T>

    Standardized Server Action response following the "never throw to client" pattern.

    Remarks

    All Server Actions return this shape instead of throwing errors. This enables predictable client-side handling with React's useActionState hook:

    const [state, action] = useActionState(myAction, null);
    if (state && !state.success) {
      // Handle error
    }
    interface ActionState<T = unknown> {
        success: boolean;
        message?: string;
        errors?: Record<string, string[]>;
        data?: T;
    }
