    Module ../Programming/Code/lease-sentinel/src/models/schema

    Domain schemas and types for LeaseSentinel.

    Remarks

    This module serves as the single source of truth for all domain types. Zod schemas provide both compile-time TypeScript types (via z.infer) and runtime validation. This ensures that data entering the system is validated at the boundary (Server Actions) and typed correctly throughout.
